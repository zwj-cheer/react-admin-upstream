# 测试策略

## 快速检查

- Vitest 覆盖运行时配置、偏好解析、响应校验、HTTP 行为、MSW 契约与能力拒绝。
- TypeScript 严格模式与 ESLint 在生产构建前运行。

## 浏览器检查

- E2E 覆盖本地登录、会话恢复/登出、只读权限、CRUD 与移动端抽屉行为。
- 视觉基线使用 1440×900、768×1024、390×844 视口并禁用动画。
- axe-core 按 WCAG A/AA 规则检查登录页、已认证外壳与对话框。

认证测试使用合成用户与一次性值。不持久化 Playwright `storageState`，并禁用
trace/video 采集，避免授权码、令牌与认证 Header 进入 CI 产物。

更新视觉基线前必须先审阅。绝不能仅因 CI 变化就接受基线。
